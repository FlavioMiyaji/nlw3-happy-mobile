# nlw3-happy-mobile
Next Level Week #3 Mobile app in ReactNative.

## Available Scripts

### `yarn start`
Starts Metro Bundler.

### `yarn android`
Run app for Android devices.

### `yarn ios`
Run app for iOS devices.

### `yarn web`
Run app for web browsers.
